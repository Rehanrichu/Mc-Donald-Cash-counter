### McDonald's Order & Billing System
# Description
This is a simple Python program that acts like a cash register for a McDonald's shop. I built this to practice the basics of Python coding.

# What it does
It asks how many burgers, fries, and colas you want.

It multiplies the items by their prices ($100 for burgers, $50 for fries, $10 for cola).

It adds everything up to show you the final bill.

If your bill is more than $600, the code tells you that you won a free dessert!

# How to use this
Run the python code
Type in numbers when asked
See your total bill and check if you won a prize
